# recipe-helper

Recipe Scaler & Merger
Problem: Cooking for 5 people using 3 different recipes (Main, Side, Salad) results in a chaotic grocery list and impossible math.\n
Failure: Recipe sites scale one recipe. None aggregate three recipes into one unified shopping list and prep schedule.\n
Scalability: A parser that takes 3 URLs, scales the ingredients, merges the shopping list, and interleaves the cooking steps (e.g., "While chicken bakes, chop the salad").
